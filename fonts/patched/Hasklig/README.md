## Installation

### Linux

- Drag the fonts from the [fonts](fonts) directory into `~/.local/share/fonts`.
- Or, use [Curl](https://github.com/curl/curl):

```sh
curl -o ~/.local/share/fonts/HaskligIF-Black.ttf https://raw.githubusercontent.com/iconicFonts/if/main/fonts/patched/Hasklig/fonts/HaskligIF-Black.ttf
curl -o ~/.local/share/fonts/HaskligIF-BlackItalic.ttf https://raw.githubusercontent.com/iconicFonts/if/main/fonts/patched/Hasklig/fonts/HaskligIF-BlackItalic.ttf
curl -o ~/.local/share/fonts/HaskligIF-Bold.ttf https://raw.githubusercontent.com/iconicFonts/if/main/fonts/patched/Hasklig/fonts/HaskligIF-Bold.ttf
curl -o ~/.local/share/fonts/HaskligIF-BoldItalic.ttf https://raw.githubusercontent.com/iconicFonts/if/main/fonts/patched/Hasklig/fonts/HaskligIF-BoldItalic.ttf

curl -o ~/.local/share/fonts/HaskligIF-LightItalic.ttf https://raw.githubusercontent.com/iconicFonts/if/main/fonts/patched/Hasklig/fonts/HaskligIF-LightItalic.ttf
```
